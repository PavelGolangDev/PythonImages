Программа написана на языке Python, в среде разработки Pyzo. Используются библиотеки для обработки изображений, такие как: PIL, numpy. В проекте реализовано изменения изображений в
чернобелый, уменьшения разрешения изображения в два раза и перевод изображения в бинарное.
Для начала работы, нужно указать путь к изображениями, в переменной path.
